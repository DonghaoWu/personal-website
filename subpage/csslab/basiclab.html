Topic 1: Length Units

Example 1:
height: 50px;
pixels (1px = 1/96th of 1in);

Example 2:
height: 5em;
1 em = current font size.;

Example 3:
height: 5rem;
The rem unit sets the font-size relative to the browsers base font-size, and will not inherit from its parents.

Example 4:
height: 50%;
Relative to the parent element.

Example 5:
height: auto;
max-width: 300px;;
auto and max-width can be used to make sure images fit tightly and snugly into available space.
